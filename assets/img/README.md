# Bildmaterial

Dieses Repository ist ausschließlich für öffentliche KlickTipp-bezogene Seiten vorgesehen.

Bildregeln:

1. nur selbst erstellte Grafiken, eindeutig lizenzierte Bilder oder ausdrücklich nutzbare Vendor-Werbemittel verwenden,
2. keine privaten Account-Screenshots ablegen,
3. keine personalisierten Werbemittel-PDFs ablegen,
4. keine Logos ohne klares Nutzungsrecht verwenden,
5. keine Testimonials ohne klare Nutzungsfreigabe verwenden,
6. Quellen- und Lizenzhinweise dokumentieren, wenn die jeweilige Lizenz das verlangt.

## Social-Preview-Bilder

Die Dateien unter `assets/img/social/` sind öffentliche Linkvorschau-Bilder für die xioo® KlickTipp Guides.

Zweck:

1. große X-Linkkarten statt kleiner Summary-Karten,
2. Open-Graph-Vorschauen für Messenger, Suchmaschinen und Social-Plattformen,
3. einheitlicher xioo® Absender für alle KlickTipp-Guides.

Regeln:

1. finale Vorschau-Bilder als PNG mit 1200 x 630 Pixeln speichern,
2. Bild-URLs in HTML immer absolut mit `https://klicktipp.xioo.de/...` setzen,
3. keine privaten Screenshots, keine Digistore24-Ansichten und keine KlickTipp-Logos verwenden,
4. bei neuen Guide-Seiten eigenes Social-Preview-Bild, `og:image`, `twitter:image` und Sitemap-Bildverweis ergänzen.
