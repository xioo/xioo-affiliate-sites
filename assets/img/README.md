# Bildmaterial

Dieses Repository ist ausschließlich für öffentliche KlickTipp-bezogene Seiten vorgesehen.

Bildregeln:

1. nur selbst erstellte Grafiken, eindeutig lizenzierte Bilder oder ausdrücklich nutzbare Vendor-Werbemittel verwenden,
2. keine privaten Account-Screenshots ablegen,
3. keine personalisierten Werbemittel-PDFs ablegen,
4. keine Logos ohne klares Nutzungsrecht verwenden,
5. keine Testimonials ohne klare Nutzungsfreigabe verwenden,
6. Quellen- und Lizenzhinweise dokumentieren, wenn die jeweilige Lizenz das verlangt.
