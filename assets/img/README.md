# Bildmaterial

Dieses Repository ist ausschließlich für öffentliche KlickTipp-bezogene Seiten vorgesehen.

Aktueller Entwurf:

1. nutzt externe Stockbilder per URL,
2. enthält keine lokalen Produktbilder,
3. enthält keine KlickTipp-Logos,
4. enthält keine Digistore24- oder KlickTipp-Account-Screenshots.

Vor produktiver Veröffentlichung prüfen:

1. ob offizielle KlickTipp-Werbemittel für öffentliche Affiliate-Landingpages freigegeben sind,
2. ob Banner oder Produktbilder nicht personalisierte Partnerdaten enthalten,
3. ob Bilddateien lokal abgelegt werden dürfen,
4. ob Quellen- oder Lizenzhinweise nötig sind,
5. ob externe Bild-URLs aus Datenschutzgründen lokal ersetzt werden sollen.

Nicht ablegen:

1. private Account-Screenshots,
2. personalisierte Werbemittel-PDFs,
3. nicht freigegebene Logos,
4. Testimonials ohne Freigabe,
5. Bilder mit unklarer Lizenz.
