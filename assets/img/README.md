# Bildmaterial

Aktueller Entwurf nutzt externe, lizenzfrei nutzbare Stockbilder per URL.

Vor produktiver Veröffentlichung prüfen:

1. ob offizielle KlickTipp-Werbemittel für öffentliche Affiliate-Landingpages freigegeben sind,
2. ob Banner oder Produktbilder nicht personalisierte Partnerdaten enthalten,
3. ob Bilddateien lokal abgelegt werden dürfen,
4. ob Quellen- oder Lizenzhinweise nötig sind.

Keine privaten Digistore24- oder KlickTipp-Account-Screenshots in dieses Verzeichnis legen.
